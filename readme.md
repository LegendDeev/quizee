# Quizee

Quizee is a modern web-based quiz application that lets users play engaging quizzes across different categories with a clean UI and smooth experience.[web:6]

Live demo: [**https://quizee-black.vercel.app/**](https://quizee-black.vercel.app/)

---

## Features

- Category-based quizzes so users can choose topics they are interested in.[web:6]
- Multiple-choice questions with automatic scoring.
- Result screen showing total score and basic summary.
- Responsive design that works on desktop and mobile browsers.[web:6]
- Deployed on **Vercel** for fast global access and easy CI/CD.
